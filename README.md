# Paystubs

## Description
System to send email with PDF from CSV with payments  

## Installation
```sh
git clone https://github.com/angelpimentell/paystubs.git
cd paystubs
cp .env.example .env
pip install -r requirements.txt
```

## Run tests
```sh
python manage.py test
```

## Usage
```sh
python manage.py runserver localhost:3000 
```

### Image folder
```sh
./static
```
